Privacy Policy for SRT AI Translator

Last Updated: April 20, 2026

Mani Stack is committed to protecting your privacy. This Privacy Policy explains how our Chrome Extension, SRT AI Translator, handles user data with transparency and security.

Core Principle: We do NOT collect, store, or transmit any personal user data to any external servers owned by the developer. Your data remains under your control at all times.

1. Data Collection & Usage

Our extension is designed to be as private as possible. We do not have a backend server to store your information, subtitles, or personal identifiers.

2. Sensitive Data Handling

  • API Keys: The API keys (Google Gemini or OpenAI) you provide are stored locally on your device's secure browser storage (chrome.storage.local). They are only used to authenticate your requests directly with the respective AI providers.
  • Subtitle Files: The content of the SRT files you upload is processed entirely within your browser environment. The text is sent directly to the AI provider you have selected for the sole purpose of translation.

3. Third-Party Services

This extension communicates directly with the following official API endpoints:

  • Google Gemini API (Google LLC)
  • OpenAI API (OpenAI, L.L.C.)

By using this extension, your subtitle text is subject to the privacy policies of the AI provider you have configured in the settings.

4. Chrome Permissions Justification

  • storage: Required to save your preferences, such as UI theme, target language, and encrypted API keys locally on your machine.
  • host permissions: Necessary to allow the extension to securely connect to official AI API servers for real-time translation.

5. Changes to This Policy

We may update this policy occasionally to reflect changes in our practices or for legal/regulatory reasons. The "Last Updated" date at the top will indicate the latest revision.
